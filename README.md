TASK-4 : 
Create a program that extracts product information, such as names, prices, and ratings, from an online e-commerce website and stores the data in a structured format like a CSV file.

Description
This project scrapes the following details from BooksToScrape.com:
Product Name
Price
Rating
The scraped data is stored in a structured CSV file named products.csv.

Technologies Used
Python
Requests
BeautifulSoup (bs4)
CSV module
VS Code

Features
Scrapes all products on the homepage
Clean extraction of text data
Converts ratings from HTML classes
Stores results in CSV format
No manual input required
